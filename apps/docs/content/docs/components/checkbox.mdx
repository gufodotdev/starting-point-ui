A Tailwind CSS checkbox component for forms.

```html preview
<input class="checkbox" type="checkbox" />
```

---

## With label

Checkboxes are typically paired with a label for accessibility.

```html preview
<div class="flex items-center gap-2">
  <input class="checkbox" type="checkbox" id="subscribe" />
  <label class="label" for="subscribe">Subscribe to newsletter</label>
</div>
```

---

## Checked state

Use the `checked` attribute to show the checkbox as selected.

```html preview
<div class="flex items-center gap-2">
  <input class="checkbox" type="checkbox" id="checked-demo" checked />
  <label class="label" for="checked-demo">This is checked</label>
</div>
```

---

## Disabled

Disabled checkboxes appear faded and cannot be interacted with. Add `peer` to the checkbox so the label also dims.

```html preview
<div class="flex flex-col gap-4">
  <div class="flex items-center gap-2">
    <input
      class="checkbox peer"
      type="checkbox"
      id="disabled-unchecked"
      disabled
    />
    <label class="label" for="disabled-unchecked">Disabled unchecked</label>
  </div>
  <div class="flex items-center gap-2">
    <input
      class="checkbox peer"
      type="checkbox"
      id="disabled-checked"
      checked
      disabled
    />
    <label class="label" for="disabled-checked">Disabled checked</label>
  </div>
</div>
```

---

## Invalid state

Checkboxes show a red ring when marked as invalid.

```html preview
<div class="flex items-center gap-2">
  <input class="checkbox" type="checkbox" id="invalid-cb" aria-invalid="true" />
  <label class="label" for="invalid-cb">You must accept the terms</label>
</div>
```
