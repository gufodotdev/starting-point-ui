A Tailwind CSS avatar component for displaying user profile images.

```html preview
<img
  class="avatar"
  src="https://images.unsplash.com/photo-1494790108377-be9c29b29330?w=150&h=150&fit=crop&crop=face"
  alt="User"
/>
```

---

## With fallback

When no image is available, display initials or an icon as a fallback.

```html preview
<span class="avatar-fallback">JD</span>
<span class="avatar-fallback"><icon-user class="size-4" /></span>
```

---

## Sizes

Avatars come in multiple sizes.

```html preview
<img
  class="avatar avatar-sm"
  src="https://images.unsplash.com/photo-1507003211169-0a1dd7228f2d?w=150&h=150&fit=crop&crop=face"
  alt="User"
/>
<img
  class="avatar"
  src="https://images.unsplash.com/photo-1507003211169-0a1dd7228f2d?w=150&h=150&fit=crop&crop=face"
  alt="User"
/>
<img
  class="avatar avatar-lg"
  src="https://images.unsplash.com/photo-1507003211169-0a1dd7228f2d?w=150&h=150&fit=crop&crop=face"
  alt="User"
/>
```

Fallbacks also scale with the size.

```html preview
<span class="avatar-fallback avatar-sm">JD</span>
<span class="avatar-fallback">JD</span>
<span class="avatar-fallback avatar-lg">JD</span>
```

## Class reference

All available classes for the avatar component.

| Class             | Description                              |
| ----------------- | ---------------------------------------- |
| `avatar`          | Base avatar styles                       |
| `avatar-fallback` | Fallback content when no image available |
| `avatar-sm`       | Small avatar                             |
| `avatar-base`     | Base avatar size (default)               |
| `avatar-lg`       | Large avatar                             |
| `avatar-group`    | Container for stacked avatar group       |
