A Tailwind CSS dropdown component for displaying a list of actions or options.

```html preview
<div class="dropdown">
  <button
    class="btn btn-outline"
    type="button"
    data-sp-toggle="dropdown"
    aria-expanded="false"
  >
    Options
  </button>
  <div
    class="dropdown-menu data-[state=open]:animate-in data-[state=open]:fade-in-0 data-[state=open]:zoom-in-95"
  >
    <a href="#" class="dropdown-item">New file</a>
    <a href="#" class="dropdown-item">Copy link</a>
    <div class="dropdown-separator"></div>
    <a href="#" class="dropdown-item">Edit</a>
    <a href="#" class="dropdown-item">Rename</a>
    <div class="dropdown-separator"></div>
    <a href="#" class="dropdown-item">Delete</a>
  </div>
</div>
```

### How it works

Use `data-sp-toggle="dropdown"` on a button inside a `.dropdown` wrapper to toggle the menu:

- Add `data-sp-toggle="dropdown"` to the trigger button
- The `.dropdown-menu` will appear when the button is clicked
- Clicking outside or pressing Escape closes the dropdown
- Clicking an item closes the dropdown

### JavaScript API

For programmatic control, use the global `sp.dropdown` module:

```js
const dropdown = document.querySelector("#my-dropdown");

sp.dropdown.open(dropdown);
sp.dropdown.close(dropdown);
sp.dropdown.toggle(dropdown);
```

## Disabled items

Use `aria-disabled="true"` to disable an item. Add `tabindex="-1"` to prevent tab navigation to the item.

```html preview
<div class="dropdown">
  <button
    class="btn btn-outline"
    type="button"
    data-sp-toggle="dropdown"
    aria-expanded="false"
  >
    Actions
  </button>
  <div
    class="dropdown-menu data-[state=open]:animate-in data-[state=open]:fade-in-0 data-[state=open]:zoom-in-95"
  >
    <a href="#" class="dropdown-item">Edit</a>
    <a href="#" class="dropdown-item" aria-disabled="true" tabindex="-1"
      >Duplicate</a
    >
    <a href="#" class="dropdown-item">Archive</a>
    <div class="dropdown-separator"></div>
    <a href="#" class="dropdown-item" aria-disabled="true" tabindex="-1"
      >Delete</a
    >
  </div>
</div>
```

## Class reference

| Class                | Description                           |
| -------------------- | ------------------------------------- |
| `dropdown`           | Container for the dropdown            |
| `dropdown-menu`      | The menu panel that appears on toggle |
| `dropdown-item`      | Individual menu item                  |
| `dropdown-label`     | Label/heading for grouping items      |
| `dropdown-separator` | Horizontal divider between items      |

---

## Data attributes

| Attribute                   | Element     | Description                                    |
| --------------------------- | ----------- | ---------------------------------------------- |
| `data-sp-toggle="dropdown"` | Button      | Toggles the dropdown                           |
| `data-sp-placement`         | `.dropdown` | Menu position (default: `bottom-end`)          |
| `data-sp-offset`            | `.dropdown` | Distance from trigger in pixels (default: `4`) |
| `aria-disabled="true"`      | Item        | Disables the item                              |

---

## Keyboard navigation

| Key               | Action                                        |
| ----------------- | --------------------------------------------- |
| `Enter` / `Space` | Opens/closes dropdown when trigger is focused |
| `Escape`          | Closes the dropdown                           |
| `ArrowDown`       | Move focus to next item                       |
| `ArrowUp`         | Move focus to previous item                   |
| `Home`            | Move focus to first item                      |
| `End`             | Move focus to last item                       |
