# Help

Our goal is to make Starting Point UI as useful as possible. If you're having trouble or have feedback, please reach out.

---

## Found a bug?

Please create an isolated, minimal code snippet that reproduces your issue and submit it via a GitHub issue.

[Submit a bug report](https://github.com/gufodotdev/starting-point-ui/issues/new)

---

## Have a feature request?

Please submit your feature request via a GitHub discussion.

[Submit a feature request](https://github.com/gufodotdev/starting-point-ui/discussions/new?category=ideas)
