# Sheet

A Tailwind CSS sheet component for slide-out panels from any edge of the screen.

```html preview
<button class="btn" data-sp-toggle="dialog" data-sp-target="#sheet-demo">
  Open Sheet
</button>

<dialog id="sheet-demo" class="dialog">
  <div
    class="sheet-backdrop data-[state=open]:animate-in data-[state=closed]:animate-out data-[state=open]:fade-in-0 data-[state=closed]:fade-out-0"
  ></div>
  <div
    class="sheet-panel data-[state=open]:animate-in data-[state=closed]:animate-out data-[state=open]:slide-in-from-left data-[state=closed]:slide-out-to-left duration-300"
  >
    <button
      class="btn btn-ghost btn-icon absolute top-4 right-4"
      aria-label="Close"
      data-sp-dismiss="dialog"
    >
      <icon-x />
    </button>
    <h2 class="text-lg font-semibold">Sheet Title</h2>
    <p class="text-muted-foreground text-sm">
      This sheet slides in from the left. Great for navigation menus or sidebars.
    </p>
  </div>
</dialog>
```

### How it works

The sheet component is custom styling on top of the [Dialog](/docs/components/dialog) component. See the dialog documentation for detailed information on opening, closing, static backdrops, and accessibility.

- Use `.sheet-panel` with an optional direction utility (`.sheet-left`, `.sheet-right`, `.sheet-top`, `.sheet-bottom`)
- The default direction is left

## Custom width

Override the default width using Tailwind utilities.

```html preview
<button
  class="btn btn-outline"
  data-sp-toggle="dialog"
  data-sp-target="#sheet-wide"
>
  Wide Sheet
</button>

<dialog id="sheet-wide" class="dialog">
  <div
    class="sheet-backdrop data-[state=open]:animate-in data-[state=closed]:animate-out data-[state=open]:fade-in-0 data-[state=closed]:fade-out-0"
  ></div>
  <div
    class="sheet-panel max-w-lg data-[state=open]:animate-in data-[state=closed]:animate-out data-[state=open]:slide-in-from-left data-[state=closed]:slide-out-to-left duration-300"
  >
    <button
      class="btn btn-ghost btn-icon absolute top-4 right-4"
      aria-label="Close"
      data-sp-dismiss="dialog"
    >
      <icon-x />
    </button>
    <h2 class="text-lg font-semibold">Wide Sheet</h2>
    <p class="text-muted-foreground text-sm">
      This sheet uses `max-w-lg` to override the default width.
    </p>
  </div>
</dialog>
```

## Class reference

| Class            | Description                               |
| ---------------- | ----------------------------------------- |
| `sheet-backdrop` | Sheet backdrop overlay                    |
| `sheet-panel`    | Base sheet panel (defaults to left)       |
| `sheet-left`     | Sheet that slides from the left (default) |
| `sheet-right`    | Sheet that slides from the right          |
| `sheet-top`      | Sheet that slides from the top            |
| `sheet-bottom`   | Sheet that slides from the bottom         |
