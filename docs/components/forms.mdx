## Input groups

Decorate inputs with icons, text, or other addons using input groups.

```html preview
<!-- START https://startingpointui.com/examples/docs/forms/2 -->
<div class="grid gap-6 w-full max-w-sm">
  <div class="input-group">
    <span class="input-group-addon"><i class="ri-search-line"></i></span>
    <input class="input" placeholder="Search..." />
  </div>
  <div class="input-group">
    <span class="input-group-addon"><i class="ri-mail-line"></i></span>
    <input class="input" type="email" placeholder="Enter your email" />
  </div>
  <div class="input-group">
    <span class="input-group-addon"><i class="ri-bank-card-line"></i></span>
    <input class="input" placeholder="Card number" />
    <span class="input-group-addon"><i class="ri-check-fill"></i></span>
  </div>
  <div class="input-group">
    <input class="input" placeholder="Rate this" />
    <span class="input-group-addon">
      <i class="ri-star-line"></i>
      <i class="ri-information-line"></i>
    </span>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/forms/2 -->
```

Input groups can include text addons for prefixes, suffixes, or units.

```html preview
<!-- START https://startingpointui.com/examples/docs/forms/3 -->
<div class="grid gap-6 w-full max-w-sm">
  <div class="input-group">
    <span class="input-group-addon">$</span>
    <input class="input" placeholder="0.00" />
    <span class="input-group-addon">USD</span>
  </div>
  <div class="input-group">
    <span class="input-group-addon">https://</span>
    <input class="input" placeholder="example" />
    <span class="input-group-addon">.com</span>
  </div>
  <div class="input-group">
    <input class="input" placeholder="Enter your username" />
    <span class="input-group-addon">@company.com</span>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/forms/3 -->
```

You can also use [Button Group](/docs/components/button) with inputs for text addons.

```html preview
<!-- START https://startingpointui.com/examples/docs/forms/4 -->
<div class="btn-group w-full max-w-sm">
  <span class="btn-group-text">https://</span>
  <input class="input" placeholder="example.com" />
</div>
<!-- END https://startingpointui.com/examples/docs/forms/4 -->
```

## Disabled state

Disabled form elements appear faded and cannot be interacted with. Add `peer` to the input so the label also dims.

```html preview
<!-- START https://startingpointui.com/examples/docs/forms/6 -->
<div class="flex flex-col gap-6 w-full max-w-sm">
  <div class="input-group">
    <span class="input-group-addon"><i class="ri-search-line"></i></span>
    <input class="input" placeholder="Searching..." disabled="" />
    <span class="input-group-addon">
      <i class="ri-loader-4-line animate-spin"></i>
    </span>
  </div>
  <input class="input" placeholder="Disabled input" disabled="" />
  <select class="select" disabled="">
    <option>Disabled select</option>
  </select>
  <textarea
    class="textarea"
    placeholder="Disabled textarea"
    disabled=""
  ></textarea>
  <div class="flex items-center gap-3">
    <input
      type="checkbox"
      class="checkbox peer"
      id="disabled-checkbox"
      disabled=""
    />
    <label class="label" for="disabled-checkbox">Disabled checkbox</label>
  </div>
  <div class="flex items-center gap-3">
    <input
      type="checkbox"
      role="switch"
      class="switch peer"
      id="disabled-switch"
      disabled=""
    />
    <label class="label" for="disabled-switch">Disabled switch</label>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/forms/6 -->
```

## Invalid state

Form elements show a red ring when marked as invalid. Use `aria-describedby` to associate error messages.

```html preview
<!-- START https://startingpointui.com/examples/docs/forms/7 -->
<div class="flex flex-col gap-6 w-full max-w-sm">
  <div class="grid gap-3">
    <div class="flex items-center justify-between">
      <label class="label" for="email-address">Email address</label>
    </div>
    <input
      class="input"
      id="email-address"
      type="email"
      aria-invalid="true"
      aria-describedby="email-address-error"
      name="email-address"
      value="johndoe"
    />
    <p id="email-address-error" class="text-xs text-destructive">
      Not a valid email address
    </p>
  </div>
  <div class="grid gap-3">
    <div class="flex items-center justify-between">
      <label class="label" for="country">Country</label>
    </div>
    <select
      class="select"
      id="country"
      name="country"
      aria-invalid="true"
      aria-describedby="country-error"
    >
      <option value="">Select a country</option>
      <option value="us">United States</option>
    </select>
    <p id="country-error" class="text-xs text-destructive">
      Please select a country
    </p>
  </div>
  <div class="flex items-center gap-3">
    <input
      type="checkbox"
      class="checkbox"
      id="invalid-cb"
      aria-invalid="true"
    />
    <label class="label" for="invalid-cb">You must accept the terms</label>
  </div>
  <div class="grid gap-3">
    <label class="label" for="amount-invalid">Amount</label>
    <div class="input-group">
      <span class="input-group-addon">$</span>
      <input
        class="input"
        id="amount-invalid"
        aria-invalid="true"
        aria-describedby="amount-error"
        value="-50"
      />
      <span class="input-group-addon">USD</span>
    </div>
    <p id="amount-error" class="text-xs text-destructive">
      Amount must be positive
    </p>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/forms/7 -->
```

## How it works

Form components are CSS-only styled native HTML elements.

### Structure

Form elements use their native HTML elements with corresponding classes:

```html
<input class="input" type="text" placeholder="Text input" />
<textarea class="textarea" placeholder="Textarea"></textarea>
<select class="select">
  <option>Option 1</option>
</select>
<input class="checkbox" type="checkbox" />
<label class="label">Label text</label>
```

For accessibility, pair inputs with labels using matching `id` and `for` attributes:

```html
<div class="space-y-2">
  <label class="label" for="email">Email</label>
  <input class="input" type="email" id="email" placeholder="you@example.com" />
</div>
```

For checkboxes, use an inline layout:

```html
<div class="flex items-center gap-2">
  <input class="checkbox" type="checkbox" id="terms" />
  <label class="label" for="terms">Accept terms</label>
</div>
```

For input groups, wrap the input with addons:

```html
<div class="input-group">
  <span class="input-group-addon"><i class="ri-search-line"></i></span>
  <input class="input" placeholder="Search..." />
</div>
```

### Class reference

All available classes for form components.

| Class               | Description                                               |
| ------------------- | --------------------------------------------------------- |
| `input`             | Add to `<input>` element for styled input                 |
| `input-sm`          | Add to `.input` for small size                            |
| `input-base`        | Add to `.input` for base size (default)                   |
| `input-lg`          | Add to `.input` for large size                            |
| `textarea`          | Add to `<textarea>` element for styled textarea           |
| `select`            | Add to `<select>` element for styled select with chevron  |
| `select-sm`         | Add to `.select` for small size                           |
| `select-base`       | Add to `.select` for base size (default)                  |
| `select-lg`         | Add to `.select` for large size                           |
| `checkbox`          | Add to `<input type="checkbox">` for styled checkbox      |
| `radio`             | Add to `<input type="radio">` for styled radio button     |
| `switch`            | Add to `<input type="checkbox">` for styled toggle switch |
| `label`             | Add to `<label>` element for styled form label            |
| `input-group`       | Add to container for grouped input styling                |
| `input-group-sm`    | Add to `.input-group` for small size                      |
| `input-group-base`  | Add to `.input-group` for base size (default)             |
| `input-group-lg`    | Add to `.input-group` for large size                      |
| `input-group-addon` | Add to addon elements (icons, text)                       |

```html
<input class="input" type="text" />
<input class="input input-sm" type="text" />
<textarea class="textarea"></textarea>
<select class="select">
  ...
</select>
<input class="checkbox" type="checkbox" />
<label class="label">Label</label>
```
